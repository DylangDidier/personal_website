Haddy Handheld Defect Logger

PythonHardware Integration3D Printing
Overview

Quality control on the factory floor used to mean juggling a camera, a clipboard, and a tape measure. I built a custom 3D-printed smart camera from scratch that does all of that in one click. An operator just points the laser at a defect, pulls the trigger, and the device snaps a picture, measures the defect, and uploads everything straight to the cloud.

My Contributions
  • Hardware & Wiring: Hooked up a high-res camera, a Time-of-Flight distance sensor, a targeting laser, and a touchscreen to a Raspberry Pi 4, getting them all to talk to each other via GPIO pins and I2C.
  • 3D Modeling & Printing: Designed the physical case from scratch using CAD and 3D printed it using tough PETG plastic so it could survive the rough environment of a factory floor.
  • Custom UI: Coded a dark-mode touchscreen interface using Python (CustomTkinter) so operators can quickly tap in the material type and what kind of defect they are logging.
  • Image Processing & Math: Wrote Python scripts that use the distance sensor to calculate exactly how big the defect is. It then slaps a "Polaroid-style" white border on the image with the measurements, timestamp, and notes baked right into the JPEG.
  • Power System & Timing: Wired in a step-down converter so the whole device runs off a standard DeWalt drill battery—if it dies, operators just hot-swap a fresh battery. I also wrote micro-delays into the code to turn off the targeting laser a split-second before the camera clicks so the red glare doesn't ruin the photo.
Results
  • Turned a really annoying, multi-step QA process into a simple, reliable point-and-shoot tool.
  • Got rid of human error when measuring defects, since the distance sensor and the Python script do all the math automatically.
  • Set it up to automatically upload logs and photos to Google Drive (using Rclone), so the engineering team can see problems on the floor in real-time.
